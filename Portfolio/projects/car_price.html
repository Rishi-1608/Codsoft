Car Price Prediction System

The Car Price Prediction System predicts the price of used cars based on various factors such as model, brand, year of manufacture, mileage, and engine size. The system is powered by a machine learning regression model to deliver accurate price predictions.

Technologies Used:

  • Python: Used for data analysis and machine learning using libraries like Pandas, Scikit-learn, and NumPy.
  • Linear Regression: To create a predictive model for car prices.
  • Flask: Used to build a web interface for users to input car details and get predictions.

Challenges:

  • Accounting for different variables like car condition and market trends.
  • Dealing with missing and inconsistent data in the dataset.

Outcome:

The system predicts car prices with an error margin of around 10%, making it a useful tool for potential car buyers and sellers.
